## Installing on Linux

For Debian and Debian derivatives including Ubuntu:

```
   sudo apt-get install libcairo2-dev
```
For Fedora:

```
   sudo yum install cairo-devel
```
For openSUSE:
```
   zypper install cairo-devel
```

```
   gcc gl_00.c -o gl_00 -lglfw3 -lGL -lcairo -lGLU -lm
```
